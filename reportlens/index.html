ReportLens

Upload CSV or JSON. Pick columns, filter rows, build charts. Everything runs in your browser — your data never leaves this page.

1 Upload data

CSV, TSV, or JSON. Max 10 MB or 50,000 rows. Files are parsed locally — nothing is uploaded.

2 Select columns

Click columns to include them in the table and exports. Type badges are inferred from the first 100 rows — click a badge to override.

Select all Select none ·

3 Filter rows expand

Add filter rules to narrow the dataset. Multiple rules combine with AND (or switch to OR). You can also tick specific rows in the table below.

4 Table view

5 Charts

Build one or more charts from the filtered data. Group by up to 4 columns — use Stack mode for cross-tab views, or Combine for one bar per unique combination.

6 Export

Downloads include only filtered rows and selected columns. Config files contain selections and filters — no data.
